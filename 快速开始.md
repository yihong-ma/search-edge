# 🚀 快速开始指南

## 最简单的方法（推荐）

### 步骤1：准备文件
确保你的项目文件夹中有以下4个文件：
- ✅ `index.html` - 主页面
- ✅ `style.css` - 样式文件  
- ✅ `app.js` - 功能代码
- ✅ `manifest.json` - 配置文件（可选）

### 步骤2：将文件传到手机

**方法A：通过微信/QQ**
1. 在电脑上，将这4个文件压缩成一个zip文件
2. 通过微信或QQ发送到手机
3. 在手机上解压文件

**方法B：通过数据线**
1. 用USB数据线连接手机和电脑
2. 将文件复制到手机存储的任意文件夹（如"下载"文件夹）

**方法C：通过云盘**
1. 将文件上传到百度网盘、OneDrive等云盘
2. 在手机上下载文件

### 步骤3：在手机上打开

1. 使用手机的文件管理器找到 `index.html` 文件
2. 点击 `index.html` 文件
3. 选择"用浏览器打开"（推荐使用Edge浏览器）
4. 如果浏览器询问是否允许弹窗，选择"允许"

### 步骤4：开始使用

1. 在打开的页面中：
   - 输入起始数字（例如：1）
   - 输入结束数字（例如：10）
   - 输入间隔时间，单位是秒（例如：2）

2. 点击"开始搜索"按钮

3. 应用会自动打开新标签页进行搜索

4. 可以随时点击"停止搜索"按钮停止

## 📱 使用示例

假设你想搜索数字1到5，每2秒搜索一次：

1. 起始数字：`1`
2. 结束数字：`5`
3. 间隔时间：`2`
4. 点击"开始搜索"

结果：应用会在新标签页中依次打开：
- 搜索"1"
- 等待2秒
- 搜索"2"
- 等待2秒
- 搜索"3"
- ...以此类推

## ⚠️ 常见问题

**Q: 点击"开始搜索"后没有反应？**
A: 检查浏览器是否阻止了弹窗。在浏览器设置中允许该网站的弹窗。

**Q: 搜索只打开一次就停止了？**
A: 可能是浏览器阻止了多个弹窗。请允许弹窗权限，或者手动允许每个弹窗。

**Q: 可以在其他浏览器中使用吗？**
A: 可以！支持所有现代浏览器（Edge、Chrome、Safari、Firefox等）。

**Q: 需要联网吗？**
A: 需要。因为搜索功能需要打开Bing搜索网站。

## 💡 提示

- 建议间隔时间至少设置为1秒，避免浏览器响应不过来
- 如果搜索范围很大，建议设置较长的间隔时间
- 可以随时点击"停止搜索"按钮停止搜索
- 搜索会在新标签页中打开，可以手动关闭不需要的标签页
