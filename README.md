# 数字搜索工具

## 项目简介

这是一个数字搜索工具，支持两种运行方式：
1. **Web版本**（推荐，无需安装任何开发工具）：直接在手机浏览器中打开使用
2. **Android应用版本**：需要Android Studio编译安装

该工具可以自动调用浏览器，按照指定的起始数字、结束数字和时间间隔，依次搜索这些数字。

## ⭐ 推荐方式：Web版本（最简单）

### 使用方法

#### 方法一：直接在手机浏览器中打开（最简单）

1. **获取文件**：
   - 将项目中的以下文件复制到你的电脑或手机：
     - `index.html`
     - `style.css`
     - `app.js`

2. **在手机上打开**：
   - **方式A**：如果文件在手机上
     - 使用文件管理器找到 `index.html` 文件
     - 点击文件，选择用浏览器打开（推荐使用Edge浏览器）

3. **开始使用**：

   - 点击"开始搜索"按钮即可

#### 方法二：通过本地服务器运行（适合开发者）

如果你有Python环境：

1. 在项目文件夹中打开命令行，运行：
```bash

# Python 2
python -m SimpleHTTPServer 8000
```

2. 确保手机和电脑在同一WiFi网络下

3. 在手机浏览器中输入：`http://电脑IP地址:8000`
   - 例如：`http://192.168.1.100:8000`

4. 打开后即可使用

#### 方法三：上传到网站服务器

如果你有自己的网站或免费托管服务（如GitHub Pages、Netlify等），可以上传这些文件，然后通过网址访问。

### Web版本使用说明

1. **打开应用**：在手机浏览器中打开 `index.html`

2. **输入参数**：
   - **起始数字**：输入要搜索的第一个数字（例如：1）
   - **结束数字**：输入要搜索的最后一个数字（例如：10）
   - **间隔时间（秒）**：输入每次搜索之间的等待时间（例如：2表示每2秒搜索一次）

3. **开始搜索**：
   - 点击"开始搜索"按钮
   - 应用会自动打开新标签页进行搜索

   - 在搜索过程中，可以随时点击"停止搜索"按钮
   - 搜索会立即停止

### Web版本注意事项

1. **浏览器弹窗**：首次使用时，浏览器可能会询问是否允许打开弹窗，请选择"允许"
2. **标签页管理**：搜索会在新标签页中打开，可以手动关闭不需要的标签页
3. **浏览器兼容性**：支持所有现代浏览器（Edge、Chrome、Safari、Firefox等）

## 参数说明

### 输入参数

| 参数名称 | 类型 | 说明 | 示例 |
|---------|------|------|------|
| 起始数字 | 整数 | 要搜索的第一个数字，必须小于等于结束数字 | 1 |
| 结束数字 | 整数 | 要搜索的最后一个数字，必须大于等于起始数字 | 10 |
| 间隔时间 | 小数 | 每次搜索之间的等待时间（秒），必须大于0 | 2.5 |

### 返回值说明

本应用不返回数据，而是通过以下方式反馈状态：

1. **状态栏显示**：显示当前搜索进度，格式为"正在搜索：当前数字 / 结束数字"
2. **Toast提示**：在操作成功或失败时显示简短提示信息
3. **按钮状态**：搜索时"开始搜索"按钮禁用，"停止搜索"按钮启用

### 核心技术

1. **开发语言**：Kotlin
2. **UI框架**：Android原生UI + Material Design组件
3. **浏览器调用**：使用Android Intent机制调用Edge浏览器
4. **定时任务**：使用Handler和Runnable实现定时搜索

### 关键代码说明

#### 浏览器调用
应用使用Intent.ACTION_VIEW来打开浏览器，优先尝试调用Edge浏览器（包名：com.microsoft.emmx），如果Edge不可用，则使用系统默认浏览器。

#### 定时搜索
使用Handler.postDelayed()方法实现定时功能，在每次搜索完成后，延迟指定时间后执行下一次搜索。

## 注意事项

1. **Edge浏览器**：应用会优先尝试使用Edge浏览器，如果手机上没有安装Edge，将使用系统默认浏览器
2. **权限要求**：应用需要网络权限和查询其他应用的权限（用于调用Edge浏览器）
3. **后台运行**：应用在后台时搜索会继续执行，如需在应用进入后台时停止搜索，可以修改MainActivity.kt中的onPause()方法
4. **数字范围**：建议不要设置过大的数字范围，以免搜索时间过长
5. **间隔时间**：建议间隔时间不要设置得太短（至少1秒），以免浏览器响应不过来

## 常见问题

### Q1: 应用无法调用Edge浏览器？
**A**: 请确保手机上已安装Microsoft Edge浏览器。如果没有安装，应用会使用系统默认浏览器。

### Q2: 搜索过程中应用崩溃？
**A**: 请检查输入的参数是否有效（数字格式正确，范围合理）。如果问题持续，请查看日志文件。

### Q3: 如何查看应用日志？
**A**: 使用Android Studio的Logcat工具，过滤标签"NumberSearch"即可查看应用日志。

### Q4: 可以修改搜索的网站吗？
**A**: 可以，在MainActivity.kt的performSearch()方法中修改searchUrl变量即可。
